# かたすみキャプチャ（KatasumiCapture）

YukkuriMovieMaker4（YMM4）向けのツールプラグインです。デスクトップ上の任意の範囲を矩形選択し、クリップボードへコピーするか、タイムラインへ画像として貼り付けるかを選べます。

**リポジトリ**: [bluemistel/KatasumiCapture](https://github.com/bluemistel/KatasumiCapture)


## 特長

- 範囲選択による画面キャプチャ（オーバーレイでドラッグ指定）
- 出力の切り替え: クリップボードへコピー / コピーして画像アイテムとして送る（タイムラインへ `Ctrl+V` 相当の貼り付け）
- 任意フォルダへの画像保存（オプション・デフォルトOFF）。保存先が空欄のときは保存せず、直接送るモードでもクリップボードのみの動作になります
- キーボードショートカット: ツール画面で「基本キー」（既定は `P`）を変更可能。`基本キー` で範囲キャプチャ、`Shift + 基本キー` で前回と同じ範囲の再キャプチャ
- 「キー操作無効」でショートカットをオフにし、画面上のボタンのみで操作可能（誤操作防止）
- 画面上部に「範囲」「再撮影」ボタン（ショートカットと同等の操作）
- 保存先はテキスト入力に加え、フォルダアイコンからディレクトリを選択可能

## 動作環境

- YukkuriMovieMaker4（YMM4）**v4.44.0 以降を想定**（.NET 10 対応の YMM4 での動作を前提としています）
- Windows 10 / 11

## インストール方法

1. 本リポジトリの **Releases** から最新の `KatasumiCapture`（または配布ファイル名が異なる場合はその `.ymme`）をダウンロードします。
 1. YMM4本体の ツール>プラグインポータル からもDL&インストールできます

YMM4 起動中は DLL がロックされ、上書きに失敗することがあります。必ず **YMM4 を終了してから** ファイルを置き換えてください。

## アンインストール方法

1. YMM4 を終了します。
2. エクスプローラーで次のフォルダを開きます。  
   `[YMM4本体のフォルダ]\user\plugin\`
3. `かたすみキャプチャプラグイン` フォルダごと削除します。
4. YMM4 を起動し、ツール一覧に「かたすみキャプチャ」が表示されないことを確認します。

## 使い方

1. YMM4 を起動し、メニューから **「かたすみキャプチャ」** ツールを開きます。
2. **出力** で「クリップボードへコピー」か「コピーして画像アイテムとして送る」のどちらかを選びます。
3. 必要に応じて **「画像も保存する」** を ON にし、保存先パスを入力するか、右のフォルダアイコンでフォルダを指定します（OFF の間は保存先欄は操作できません）。
4. キャプチャの開始方法は次のいずれかです。  
   - 画面上部の **「範囲」** ボタン、または YMM4 がアクティブなときに設定した **基本キー**  
   - 前回と同じ矩形で撮り直す場合は **「再撮影」** ボタン、または **Shift + 基本キー**
5. 範囲選択モードでは画面が暗く表示されるので、ドラッグで矩形を指定し、マウスボタンを離すと確定します。

### ショートカットと誤操作防止

- ショートカットは **YMM4 のウィンドウがアクティブなときのみ** 反応します。
- ショートカットキーは **基本キー（A〜Z, 0〜9）** と **Shift** の組み合わせで動作します。
- **「キー操作無効」** にチェックを入れると、キーボードショートカットは無効になります（ボタン操作は可能）。
- YMM4内でテキストを打ち込みたい場合は **「キー操作無効」** にチェックを入れてお使いください。

### タイムラインへ送るモードについて

画像はクリップボードに載せたうえで、本プラグインから `Ctrl+V` を送って YMM4 側の貼り付け処理に任せています。フォーカスがタイムライン側にない場合、期待どおりに貼り付けられないことがあります。その場合はタイムラインをクリックしてから再実行してください。
