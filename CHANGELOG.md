# 変更履歴

このファイルの形式は [Keep a Changelog](https://keepachangelog.com/ja/1.0.0/) に近づけています。

## [1.0.1] - 2026-03-19

### Fixed

- ダークモード時も注意書きテキストがテーマに追従するよう、固定色をやめてホストの前景色を継承するよう変更

### Added

- 範囲選択オーバーレイ表示中に **Esc** キーでキャプチャを中断できるようにした

### Changed

- **キー操作無効** の既定を **ON** に変更（初回起動時からショートカット無効）

## [1.0.0] - 2026-03-19

### Added

- 矩形範囲の画面キャプチャ
- クリップボードへコピー / タイムラインへ画像として送る（貼り付け）の切り替え
- 任意フォルダへの保存（オプション）、フォルダ選択ダイアログ
- キーボードショートカット（基本キー・Shift 組み合わせ、YMM4 アクティブ時のみ）
- キー操作無効、画面上の「範囲」「再撮影」ボタン
- 初版として GitHub 公開用の README・ライセンス・利用規約の整備
